### 프로젝트 설명

해당 프로젝트는 React와 Redux Toolkit을 이용하여 만든 상점페이지입니다.

#### 프로젝트의 주요 기능

- swiper를 이용한 터치 슬라이드 구현
- redux를 이용한 실시간 반영되는 장바구니 상품의 수
- 상품 이름을 이용한 검색 기능
- 다크모드 & 화이트 모드
